## [[Корреляционная матрица]]

Достроим таблицу для нахождения законов распределения X и Y c целью найти их числовые характеристики

| $X$\\$Y$          | 0             | 1             | 2              | $\sum \rightarrow$ |
| ----------------- | ------------- | ------------- | -------------- | ------------------ |
| 0                 | $\frac{1}{4}$ | $\frac{1}{3}$ | $\frac{1}{9}$  | $\frac{25}{36}$    |
| 1                 | 0             | $\frac{1}{6}$ | $\frac{1}{9}$  | $\frac{5}{18}$     |
| 2                 | 0             | 0             | $\frac{1}{36}$ | $\frac{1}{36}$     |
| $\sum \downarrow$ | $\frac{1}{4}$ | $\frac{1}{2}$ | $\frac{1}{4}$  |                    |

Получаем законы распределения составляющих:

| $X$ | 0               | 1              | 2              |
| --- | --------------- | -------------- | -------------- |
| $p$ | $\frac{25}{36}$ | $\frac{5}{18}$ | $\frac{1}{36}$ |

| $Y$ | 0             | 1             | 2             |
| --- | ------------- | ------------- | ------------- |
| $p$ | $\frac{1}{4}$ | $\frac{1}{2}$ | $\frac{1}{4}$ |

Их [[Математическое ожидание случайной величины|матожидания]]:

$M[X] = \frac{1}{3}$

$M[Y] = 1$

Для вычисления [[Дисперсия случайной величины|дисперсии]] по [[Сокращенная формула дисперсии|сокращенной формуле]], требуется закон распределения для квадратов компонент:

| $X$ | 0               | 1              | 4              |
| --- | --------------- | -------------- | -------------- |
| $p$ | $\frac{25}{36}$ | $\frac{5}{18}$ | $\frac{1}{36}$ |

| $Y$ | 0             | 1             | 4             |
| --- | ------------- | ------------- | ------------- |
| $p$ | $\frac{1}{4}$ | $\frac{1}{2}$ | $\frac{1}{4}$ |

[[Математическое ожидание случайной величины|матожидания]]:

$M[X^2] = \frac{7}{18}$

$M[Y^2] = \frac{3}{2}$

[[Дисперсия случайной величины|дисперсии]]

$D[X] = M[X^2] - M^2[X] = \frac{5}{18}$

$D[Y] = M[Y^2] - M^2[Y] = \frac{1}{2}$

[[Стандартное отклонение случайной величины|стандартные отклонения]]

$\sigma_x = \sqrt{D[X]} \approx  0.5270$

$\sigma_y = \sqrt{D[y]} \approx  0.7071$

Модифицируем таблицу для нахождения [[Ковариация случайных величин|ковариации]], уменьшив на соответствующие матожидания первую строку и первый столбец

| $X$\\$Y$       | -1            | 0             | 1              |
| -------------- | ------------- | ------------- | -------------- |
| $-\frac{1}{3}$ | $\frac{1}{4}$ | $\frac{1}{3}$ | $\frac{1}{9}$  |
| $\frac{2}{3}$  | 0             | $\frac{1}{6}$ | $\frac{1}{9}$  |
| $\frac{5}{3}$  | 0             | 0             | $\frac{1}{36}$ |

Тогда ковариация из формулы [[Теорема. Ковариация равна матожиданию произведения центрированных случайных величин|теоремы]] вычисляется как сумма произведений случайных величин на вероятность их наступления:

$K_{xy} = \frac{1}{6}$

Запишем матрицу:

$$
\begin{pmatrix}
    \frac{5}{18} & \frac{1}{6} \\
    \frac{1}{6} & \frac{1}{2} \\
  \end{pmatrix}
$$

## [[Коэффициент корреляции]]

вычислим по определению
$r_{xy}=\frac{K_{xy}}{\sigma_x\sigma_y} \approx 0.4472$

## [[Функция распределения случайной величины]]

![[Pasted image 20250215190003.png]]

Суммируем вероятности тех точек, которые попали в рассматриваемый бесконечный прямоугольник

| $X$\\$Y$       | $y\leq -1$ | $-1 < y \leq 0$ | $0 < y \leq 1$ | $1 < y$ |
| -------------- | ---------- | --------------- | -------------- | ------- |
| $x \leq 0$     | 0          | 0               | 0              | 0       |
| $0 < x \leq 1$ | 0          | 0.06            | 0.16           | 0.2     |
| $1 < x \leq 2$ | 0          | 0.27            | 0.72           | 0.9     |
| $2 \leq x$     | 0          | 0.3             | 0.8            | 1       |
