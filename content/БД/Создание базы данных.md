Информация о каждой БД хранится в таблице *sysdatabases* базы данных *master*.

При создании БД, нужно делать резервную копию master

```sql
CREATE DATABASE example db;
```

Инструкция выполняется в режиме автоматической фиксации [[Базы данных. Определения#режим автоматической фиксации|режиме автоматической фиксации]]

Не может выполняться в прочих транзакциях

![[Pasted image 20250103074817.png]]

В msql может быть не более 32767 БД

У каждой бд есть владелец - пользователь, создавший БД

Владельца можно изменить, выполнив процедуру sp_changedbowner

При создании бд создается 2 файла: dbname и dbname_log
