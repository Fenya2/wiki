*Явным [[k-шаговые методы Адамса для решения задачи Коши|методом]] Адамса $k$-го порядка* называется метод численного интегрирования, использующий [[Интерполяционный многочлен Лагранжа|интерполяционный многочлен Лагранжа]] $L_{k-1}(x)$, построенный по значениям правой части $f(x, y)$ в $k$ предыдущих точках. $k$ может принимать значения $0,1,\dots, n-1$. Формула имеет вид:

$$
y_{m+1} = y_m + \int_{x_m}^{x_{m+1}} L_{k-1}(x) \, dx, \quad m = k - 1, k, \ldots, n - 1
$$

$L_{k-1}(x)$ - интерполяционный многочлен Лагранжа степени $k-1$, построенный по предыдущим приближенным точкам $(x_{m-j}, f(x_{m-j}, y_{m-j}))$ для $j = 0, 1, \dots, k-1$
