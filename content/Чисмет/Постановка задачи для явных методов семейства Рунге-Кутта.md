# Вводная

Методы [[Метод Эйлера с пересчетом|Эйлера с пересчетом]] и [[Метод Коши|Коши]] объединяет схожая 2-х шаговая процедура при рассчете очередного приближенного значения решения [[Постановка задачи Коши для дифференциального уравнения 1 порядка|задачи Коши]].

Обобщим это наблюдение, выделив семейство методов, в которых на каждом шаге требуется последовательно вычислять **два** значения правой части ($f$) дифференциального уравнения:

$y_{i+1} = y_i + p_1 K_1 + p_2 K_2$

1) $K_1 = h f(x_i, y_i),$
2) $K_2 = h f(x_i + \alpha h, y_i + \beta K_1)$

- $i = 0, 1, 2, \ldots, n-1$
- $p_1, p_2, \alpha, \beta \in \mathbb{R}$

Явный метод Эйлера также принадлежит семейству

| Метод                         | $p_1$         | $p_2$         | $\alpha$      | $\beta$       |
| ----------------------------- | ------------- | ------------- | ------------- | ------------- |
| [[Явный метод Эйлера]]        | $1$           | $0$           | любой         | любой         |
| [[Метод Эйлера с пересчетом]] | $\frac{1}{2}$ | $\frac{1}{2}$ | $1$           | $1$           |
| [[Метод Коши]]                | 0             | 1             | $\frac{1}{2}$ | $\frac{1}{2}$ |

Это семейство методов называется *семейством явных методов Рунге-Кутты*
# Постановка
Требуется найти такие параметры $p_1, p_2, \alpha, \beta$ такие, чтобы метод, составленный по этим параметрам имел 2 [[Теорема. Явный метод Эйлера имеет 1 порядок точности на всей области решения задачи Коши|порядок точности на всей области решения]] $[x_0, x_0 + X]$. Для этого, в силу [[Замечание. Связь между порядком точности итерационного метода на одном шаге и на всей области решения|замечания]], методы должны обеспечивать 3 [[Метод решения задачи Коши, основанный на разложении функции в ряд Тейлора|порядок точности на одном шаге]]
