Решается [[Постановка задачи нахождения оптимальных узлов для интерполяционной квадратурной формулы с весовой функцией|задача нахождения интерполяционной квадратурной формулы]] с максимальной [[Алгебраическая степень точности квадратурной формулы|алгебраической степенью точности]]
# Вход
- [[Постановка задачи вычисления определенных интегралов с весовой функцией|Задача]]
	- с ограничениями на весовую функцию $p$, чтобы удовлетворять [[Теорема. Критерий максимальной алгебраической степени точности интерполяционной квадратурной функции с весовой функцией|критерию]]
	- c $n+1$ [[Квадратурная сумма|узлом]] для выбора (еще не зафиксированы)
# Выход
[[Интерполяционная квадратурная формула с весовой функцией]] c алгебраической степенью точности $N \ge 2n+1$, построенная по выбранным в процессе работы процедуры узлам
# Численная процедура
1) Согласно [[Теорема. О существовании многочлена ортогонального всем многочленам степени n с весом|теореме]] строится неоднородная система линейных алгебраических уравнений относительно неизвестных коэффициентов $a_i, i = 1, 2, \dots, n + 1$ многочлена $\omega_n$
2) Находится решение $\bar{a}_1, \bar{a}_2, \dots, \bar{a}_{n+1}$ системы и определяется многочлен
$$
\omega_n(x) = x^{n+1} + \bar{a}_1 x^n + \bar{a}_2 x^{n-1} + \dots + \bar{a}_{n-1} x^2 + \bar{a}_n x + \bar{a}_{n+1}
$$
3) Вычисляются корни $\bar{x}_i$ ($i = 0, 1, 2, \dots, n$) многочлена $\omega$
4) В силу [[Интерполяционная квадратурная формула с весовой функцией|интерполяционности]] искомой квадратурной формулы по её формулам вычисляются [[Квадратурная сумма|квадратурные коэффициенты]] квадратурной суммы исходной формулы
5) Строится квадратурная интерполяционная формула с алгебраической степенью точности $N \geq 2n + 1$ в силу [[Теорема. Критерий максимальной алгебраической степени точности интерполяционной квадратурной функции с весовой функцией|критерия]]
