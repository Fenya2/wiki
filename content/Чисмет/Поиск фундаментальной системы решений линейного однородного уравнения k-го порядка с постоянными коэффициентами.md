Из [[Свойство. Структура общего решения линейного однородного разностного уравнения|свойства]] следует, что нужно найти Фундаментальную систему линейно-незавиисимых решений для нахождения общего решения [[Линейное однородное разностное уравнение k-го порядка с постоянными коэффициентами|ЛОРУkПсПК]]

Для нахождения фундаментальной системы решений уравнения предлагается искать его линейно-независимые решения в виде $y_s = \rho^s$ (взяли вид решения из воздуха?)

Подстановка $y_m = \rho^m$ в уравнение приводит к уравнению вида:

$$
\rho^m + a_1 \rho^{m-1} + a_2 \rho^{m-2} + \dots + a_k \rho^{m-k} = 0,
$$

После сокращения на $\rho^{m-k}$, получаем алгебраическое уравнение, называемое *характеристическим* для рассматриваемого [[Линейное однородное разностное уравнение k-го порядка с постоянными коэффициентами|ЛОРУkПсПК]]

$$
\rho^k + a_1 \rho^{k-1} + a_2 \rho^{k-2} + \dots + a_{k-1} \rho + a_k = 0
$$
> by the way. Уравнение является многочленом
