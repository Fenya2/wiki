Для всех методов, решающих [[Постановка задачи Коши для дифференциального уравнения 1 порядка|задачу Коши]] и использующих [[Метод решения задачи Коши, основанный на разложении функции в ряд Тейлора|идею с разложением функции в ряд Тейлора]] или на [[Метод решения задачи Коши, основанный на интегральном представлении решения|идее интегрального представления решения]] справедлива следующая закономерность:

**Если [[Метод решения задачи Коши, основанный на разложении функции в ряд Тейлора|порядок точности метода на одном шаге]] равен $k+1$, то [[Теорема. Явный метод Эйлера имеет 1 порядок точности на всей области решения задачи Коши|порядок точности этого метода на всей области решения]] равен $k$**

Это подробнее доказывается для $k = 1$ [[Теорема. Явный метод Эйлера имеет 1 порядок точности на всей области решения задачи Коши|здесь]], но, по аналогии, доказывается для любого $k$
