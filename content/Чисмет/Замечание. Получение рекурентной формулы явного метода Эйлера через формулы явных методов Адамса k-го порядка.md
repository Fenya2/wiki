Рекурентная формула [[Явный метод Эйлера|явного метода Эйлера]] может также быть получена с использованием формул [[Явный метод Адамса k-го порядка|явных методов Адамса k-го порядка]] при $k=1$ для решения [[Постановка задачи Коши для дифференциального уравнения 1 порядка|задачи Коши]]:

$$
L_0(x) = F(x_m) \overset{def}{=} f(x_m, y_m)
$$

Строится [[Интерполяционный многочлен Лагранжа|интерполяционный многочлен Лагранжа]] степени 0 по единственному предыдущему узлу $x_m, m=0,1,\dots n-1$

Тогда конкретная явная формула Адамса 1 порядка имеет вид

$$
y_{m+1} = y_m + \int_{x_m}^{x_{m+1}} L_0(x)\,dx \overset{(1)}{=}
y_m + f(x_m, y_m) h,
\quad m = 0, 1, \dots, n - 1.
$$

1 - расписали многочлен $L_0$ по определению выше. Он представляет собой константу. Вынесли ее из под интеграла, исползуя свойство однородности интеграла. Полученный интеграл равняется 1 (разность между концами отрезка интегрирования составляет размер шага $h$)
