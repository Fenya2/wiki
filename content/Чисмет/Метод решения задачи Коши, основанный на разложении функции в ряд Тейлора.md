Пусть $f$ в [[Постановка задачи Коши для дифференциального уравнения 1 порядка|задаче]] выбрана так, что у точного решения задачи $y(x|_{x_0,y_0})$ существуют производные до достаточно большого порядка. Считаем, что в искомом численном решении узлы будут равностоящими, то есть расстояние между соседними равняется $h = \frac{X}{n}$ (*размер шага*):

$$
x_{i+1} = x_i + h, \forall i = 0, \dots, (n-1)
$$

Разложим решение $y(x|_{x_0,y_0})$ в ряд Тейлора в окрестности [[Постановка задачи Коши для дифференциального уравнения 1 порядка|узла]] $x_0$ (с приращением $h$). Остаточный член запишем в форме Пеано:

$$
y(x_0 + h) = y(x_0) + y'(x_0)h + \frac{y''(x_0)}{2!}h^2 + \dots + \frac{y^{(k)}(x_0)}{k!}h^k + O(h^{k+1})
$$

В зависимости от рассматриваемого уравнения и требуемой точности, формулу можно записывать для различных $k$. Возьмем, в качестве примера разложение до б.о.о 2 члена ($k=2$):

$$
y(x_0 + h) \overset{(1)}{=}
y(x_0) + f(x_0, y(x_0))h + \frac{h^2}{2}\left[f'_x(x_0, y(x_0)) + f'_y(x_0, y(x_0))f(x_0, y(x_0))\right] + O(h^3)
$$

$y'$ заменили на $f$ по условию [[Постановка задачи Коши для дифференциального уравнения 1 порядка|задачи Коши]]:
- $y'(x_0) = f(x_0, y(x_0))$
- $y''(x_0) = f'_x(x_0, y(x_0)) + f'_y(x_0, y(x_0))y'(x_0) = f'_x(x_0, y(x_0)) + f'_y(x_0, y(x_0))f(x_0, y(x_0))$

Отбрасывание остаточного члена приводит к приближенному равенству

$$
y(x_0 + h) \approx
y(x_0) + f(x_0, y(x_0))h + \frac{h^2}{2}\left[f'_x(x_0, y(x_0)) + f'_y(x_0, y(x_0))f(x_0, y(x_0))\right]
$$

которое связывает точное значение $y(x_0)$ c приближенным значение $y(x_1) = y(x_0 + h)$

***Требование точности этого равенства для приближённых значений*** $y_i$ и $y_{i+1}$ в произвольных двух соседних узлах $x_i$ и $x_{i+1}$ $(0 \leq i < n)$ приводит к реккурентной формуле:

$$
y_{i+1} = y_i + f(x_i, y_i)h + \frac{h^2}{2} \left( f'_x(x_i, y_i) + f'_y(x_i, y_i) f(x_i, y_i) \right)
\quad \forall i = 0, 1, \ldots, n - 1,
$$

которая задаёт итерационный метод <mark style="background: #FFB86CA6;">3-го порядка точности на одном шаге</mark> (итерации), так как

$$
|y(x_1) - y_1| = O(h^3), i = 0.
$$

Аналогично работает идея решени при других значениях $k$
