# Пусть

- $f \in C^{(2n+2)}([a, b])$
- Для $f$ составлена [[Квадратурная формула Гаусса|квадратурная формула гаусса]] с вытекающими из [[Теорема. Критерий максимальной алгебраической степени точности интерполяционной квадратурной функции с весовой функцией|критерия]] ограничениями на вид весовой функции $p$

# Тогда

$$
R_n[f] = \frac{f^{(2n+2)}(\xi)}{(2n + 2)!} \int_a^b p(x) (\omega_n(x))^2 dx
$$

и справедлива оценка

$$
|R_n[f]| \leq \frac{M_{2n+2}}{(2n + 2)!} \int_a^b p(x) (\omega_n(x))^2 dx,
$$

- $\xi \in [a, b]$
- $M_{2n+2} = \max_{x \in [a, b]} |f^{(2n+2)}(x)|$

# Доказательство

Пусть $L_{2n+1}(x)$ - [[Интерполяционный многочлен Эрмита|интерполяционный многочлен Эрмита]], построенный для функции $f$ по набору двукратных узлов

$x_i \in [a, b]$ ($i = 0, 1, \dots, n$)

Тогда, в силу [[Свойство. Разделенная разность для k раз непрерывно-дифференцируемых функций|свойства]](?)

$$
f(x) = L_{2n+1}(x) + f(x, x_0, x_0, \dots, x_n, x_n) (\omega_n(x))^2.
$$

Следовательно, из [[Постановка задачи вычисления определенных интегралов с весовой функцией|постановки задачи]], погрешность считается как:

$$
R_n[f] = \int_a^b p(x)f(x, x_0, x_0, \dots, x_n, x_n) (\omega_n(x))^2 dx
= f(\eta, x_0, x_0, \dots, x_n, x_n) \int_a^b p(x) (\omega_n(x))^2 dx
= \frac{f^{(2n+2)}(\xi)}{(2n+2)!} \int_a^b p(x) (\omega_n(x))^2 dx
$$

- $\eta \in [a, b]$ (по теореме о среднем)
- $\xi \in [a, b]$
