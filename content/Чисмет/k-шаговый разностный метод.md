Пусть решается [[Постановка задачи Коши для дифференциального уравнения 1 порядка|задача Коши]]

*$k$-шаговым разностным методом* решения задачи называется рекурентная формула

$$
y_m = \sum_{i=1}^k \alpha_i y_{m-i} + h \sum_{j=0}^k \beta_j f_{m-j}
\tag{1}
$$

- $m = k, \dots, n - 1$
- $\alpha_i \in \mathbb{R}$ ($i = 1, 2, \dots, k$)
- $\beta_j \in \mathbb{R}$ ($i = 1, 2, \dots, k$)
- $f_{m-j} = f(x_{m-j}, y_{m-j})$ ($j = 0, 1, \dots, k$)

Говорят, что формула $(1)$ *точна*, если она находит точные значения [[Постановка задачи Коши для дифференциального уравнения 1 порядка|решения]] во всех рассматриваемых узлах ($y_i = y(x_i), i = 0,\dots,n-1$)
