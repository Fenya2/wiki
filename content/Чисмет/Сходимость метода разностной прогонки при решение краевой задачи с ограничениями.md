[[Применение метода разностной прогонки для решения краевой задачи с ограничениями|Применили]] [[Метод разностной прогонки|метод разностной прогонки]] для решения [[Постановка краевой задачи с ограничениями на f и граничные условия|краевой задачи]]

Возникает вопрос, как будет вести себя приближенное решение, рассчитанное методом, с уменьшением размера шага $h$

Покажем, что при $h \rightarrow 0$ соответвующее приближенное решение $y_i(h) \rightarrow y(x_i)$ (приближенное решение будет сходится к точному решению задачи)

Для этого достаточно оценить погрешности

$$
\varepsilon_i = y(x_i) - y_i,\quad i = 0,1,\dots,n
$$

# Дифференциальное уравнение

[[Сведение краевой задачи с ограниченями к системе линейных алгебраических уравнений|Помним]], что 

$$
y''(x) = \frac{y(x-h) - 2y(x) + y(x+h)}{h^2} + R_i,\quad i = 1,\dots,n-1
\tag{1}
$$

Причем

$$
R_i = - \frac{h^2}{12} y^{(4)}(\xi(x))
$$

- $ξ(x) ∈ [x − h, x + h]$
- $i = 1,\dots,n-1$

и справедлива простая оценка:

$$
|R_i| = |\frac{h^2}{12} y^{(4)}(\xi(x))| \le \frac{M_4h^2}{12}
$$

- $M_4 = max_{x\in [a,b]}|y^{(4)}(x)|$
- $i = 1,\dots,n-1$

Подставим $(1)$ в [[Постановка краевой задачи с ограничениями на f и граничные условия|исходное дифференциальное уравнение]]:

$$
\frac{y(x-h) - 2y(x) + y(x+h)}{h^2} + R_i = p(x_i)y(x_i) + q(x_i),\quad i = 1,\dots,n-1
$$

Вычтем из равенства равенство, полученное после [[Сведение краевой задачи с ограниченями к системе линейных алгебраических уравнений#^409fe6|дискретизации]], когда сводили задачу к системе линейных алгебраических уравнений. Получим

$$
\frac{(y(x_{i-1}) - y_{i-1}) - 2(y(x_i) - y_i) + (y(x_{i+1})-y_{i+1})}{h^2} + R_i = p(x_i)(y(x_i) - y_i),\quad i = 1,\dots,n-1
$$

В терминах погрешностей равенство можно переписать в виде

$$
\frac{\varepsilon_{i-1} - 2\varepsilon_{i} + \varepsilon_{i+1}}{h^2} + R_i = p(x_i)\varepsilon_{i},\quad i = 1,\dots,n-1
$$

Преобразовав уравнение, получим пачку $(n-1)$ привычных линейных неоднородных уравнений:

$$
\varepsilon_{i-1} - (2+h^2p_i)\varepsilon_{i} + \varepsilon_{i+1} = -R_ih^2,\quad i = 1,\dots,n-1
\tag{2}
$$

# Краевые условия

## Первого рода

В случае краевых условий 1 рода $\varepsilon_0 = y(x_0) - y_0 = 0, \varepsilon_n = y(x_n) - y_n = 0$, положили это [[Сведение краевой задачи с ограниченями к системе линейных алгебраических уравнений#Краевых условия 1 рода|тут]].

Получили вместе с $(2)$ систему линейных уравнений:

$$
\begin{cases}
\varepsilon_{0} = 0\\
\varepsilon_{i-1} - (2+h^2p_i)\varepsilon_{i} + \varepsilon_{i+1} = -R_ih^2,\quad i = 1,\dots,n-1\\
\varepsilon_{n} = 0
\end{cases}
$$

Рассмотрим уравнения $i=1,\dots,n-1$. Они преобразуются к виду и оцениваются:

$$
(2+h^2p_i)\varepsilon_{i} = \varepsilon_{i-1} + \varepsilon_{i+1} + R_ih^2 \le 2|\varepsilon^*| + |R_i|h^2
$$

$\varepsilon^* = \max_{i=1,\dots,n-1}|\varepsilon_i|$

Неравенство не изменится, если сделать замену:

$$
(2+h^2p^*)|\varepsilon^*| \le 2|\varepsilon^*| + |R^*|h^2
$$

- $p^* = p(x^*), x^*$-узел, которому соответсувует $\varepsilon^*$
- $R^*$ - погрешность для $\varepsilon^*$

Приведя подобные получим:

$$
h^2p^*|\varepsilon^*| \le |R^*|h^2
$$

Откуда

$$
|\varepsilon^*| \le \frac{|R^*|}{p^*}
$$

С учетом оценки $R^*$ и [[Постановка краевой задачи с ограничениями на f и граничные условия|свойств]] p, получаем оценку:

$$
|y(x_i) -y_i| \le \frac{M_4h^2}{12\overline{p}},\quad i = 0,1,\dots,n
$$

**То есть метод разностной прогонки в случае краевых условия 1 рода сходится при решении задачи со вторым порядком по $h$**

## Третьего рода

Здесь нужно дополнительно учитывать погрешность формул численного дифференцирования для первой производной на краях:

$$
y'(a) = \frac{y_1 - y_0}{h} + r_0
$$

$$
y'(b) = \frac{y_n - y_{n-1}}{h} + r_n
$$

$$
|r_0| \leq \frac{1}{2}M_2h, \quad |r_n| \leq \frac{1}{2}M_2h
$$

$M_2 = \max_{x \in [a,b]} |y''(x)|$

Действуя по аналогии с оценкой погршности при условиях первого рода получаем систему:

$$
\begin{cases}
- (1 + h\alpha_0) \varepsilon_0 + \varepsilon_1 = r_0\\
\varepsilon_{i-1} - (2 + h^2 p_i) \varepsilon_i + \varepsilon_{i+1} = -R_i h^2 & i=1,\dots,n-1 \\
\varepsilon_{n-1} - (1 + h\beta_0) \varepsilon_n = -r_n &
\end{cases}
$$

Действуй по аналогии с оценкой погрешностей с краевыми условиями 1 рода, получаем оценку:

$$
|y(x_i) - y_i| \leq \max\left\{
\frac{1}{2}M_2h,\ 
\frac{1}{2}M_2h,\ 
\frac{1}{12}M_4h^2
\right\}, \quad i = 0,1,2,\dots,n
$$

- $M_2 = \max\limits_{x\in[a,b]}|y''(x)|$
- $M_4 = \max\limits_{x\in[a,b]}|y^{(4)}(x)|$

**То есть метод разностной прогонки в случае краевых условия 3 рода сходится при решении задачи с первым порядком по $h$
