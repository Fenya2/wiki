[[Постановка краевой задачи|Краевую задачу]] можно свести к 2 связанным подзадачам:

1) [[Постановка задачи Коши для дифференциального уравнения k-го порядка|задача Коши для дифференциального уравнения 2 порядка]]
	- $y'' = f(x, y, y'), \quad x \in [a, b]$
	- $y(a) = \alpha$
	- $y'(a) = \mu$
2) Уравнение
	- $\phi(\mu) \overset{def}{=} y_{\mu}(b) - \beta = 0$ - реально по сути функция от $\mu$

Исходная задача будет решена, если найдется такой параметр $\mu^*$, что решение задачи Коши $y_{\mu^*}$ с этим параметром при подстановке в уравнение даст верное равенство

Задачу Коши $(1)$ при фиксированном $\mu$ можно решать численными методами (после сведения к "системе" с 2 задачам [[Постановка задачи Коши для дифференциального уравнения 1 порядка|задачами Коши для дифференциальных уравнений 1 порядка]]) как отдельную задачу

> Тут важно осознать, что $y_{\mu}$ может рассматриваться как функция двух переменных: $y(x, \mu)$

После нахождения решения задачи Коши, как отдельную задачу можно решить уравнение $(2)$, также численно, уточнив искомый параметр $\mu$

Уточнять параметр нужно до тех пор (перерешивая задачу Коши и уравнениие), пока найденное решение не совпдет с решением исходной задачи или не точность решения не будет удовлетворять заранее установленной погрешности

Эта методика решения краевой задачи называется *методом стрельбы*

Задача Коши $(1)$ в методе называется *внутренней задачей метода стрельбы*
Поиск решения уравнения $(2)$ в методе называется *внешней задачей метода стрельбы*

То есть в терминах метода стрельбы исходная задача сводится к поиску такого параметра $\mu^*$, чтобы решение внутренней задачи являлось решением исходной задачи
