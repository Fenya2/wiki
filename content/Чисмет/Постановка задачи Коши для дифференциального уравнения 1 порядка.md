# Дано

дифференцильное уравнение первого порядка

$$
y' = f(x,y)
$$

- На отрезке $[x_0, x_0 + X], X \ge x_0$ - *область решения задачи*
- $f$
	- Непрерывна по каждому своему аргументу
	- Имеет непрерывную частную производную $f'_y$

Причем известно, что (начальные условия)

$$
y(x_0) = y_0
$$

# Требуется

Отыскать решение дифференциального уравнения, с учетом начальных условий - **предоставить набор $n$ пар**

$$
(x_i, y_i), i = 0, \dots (n-1), x_i \in [x_0, x_0 + X], x_{i+1} = x_i + h_i, y_i \approx y(x_i)
$$

$h_i > 0 \forall i=0, \dots, (n-1)$

- $x_i, i = 0, \dots\, (n-1)$ назовем *узлами*
- $y_i, i = 0, \dots\, (n-1)$ назовем *приближенными значениями решения в узле* $x_i$

Такой набор пар также называется *сеточной функцией*

> Почти всегда при рассмотрении методов $h_i, i = 0, \dots, n-1$ равны между собой. Если не говорится обратное, полагать, что так оно и есть и считать $\forall i=0,\dots,n-1\,\, h_i = h = \frac{b-a}{n}$
