Аналогично [[Замечание. Получение рекурентной формулы явного метода Эйлера через формулы явных методов Адамса k-го порядка|замечанию]]

Рекурентная формула [[Неявный метод Эйлера|неявного метода Эйлера]] может также быть получена с использованием формул [[неявный метод Адамса k-го порядка|неявных методов Адамса k-го порядка]] при $k=0$ для решения [[Постановка задачи Коши|задачи Коши]]:

Строится [[Интерполяционный многочлен Лагранжа|интерполяционный многочлен Лагранжа]] степени 0 по единственному текущему узлу $x_m, m=0,1,\dots n-1$

$$
L_0(x) = F(x_{m+1}) = f_{m+1},
$$

Получаем формулу
$$
y_{m+1} = y_m + \int_{x_m}^{x_{m+1}} L_0(x) \, dx
= y_m + f_{m+1} h =
y_m + f(x_{m+1}, y_{m+1})h
$$

$m = 0, 1, \dots, n - 1$
