[[Метод Эйлера с пересчетом]] обладает 2 [[Теорема. Явный метод Эйлера имеет 1 порядок точности на всей области решения задачи Коши|порядком точности на всей области решения]] [[Постановка задачи Коши для дифференциального уравнения 1 порядка|задачи Коши]]

# Доказательство

Покажем, что метод обладает 3 [[Метод решения задачи Коши, основанный на разложении функции в ряд Тейлора|порядком точности итерационного метода на одном шаге]]

Процедура метода выглядит следующим образом:

1) $\bar{y}_{k+1} = y_k + h f(x_k, y_k)$ - нашли "промежуточное" решение [[Явный метод Эйлера|явным методом Эйлера]] (1)
2) $y_{k+1} = y_k + \frac{h}{2} \left[ f(x_k, y_k) + f(x_{k+1}, \bar{y}_{k+1}) \right]$ "пересчитали", подставив в формулу (2)

$k = 0, 1, 2, \ldots, n - 1$

Для определения порядка погрешности метода на одном шаге используется следующее разложение функции $y(x|_{x_0, y_0})$ в ряд Тейлора в окрестности узла $x_0$:

$$
y(x_1) = y(x_0) + f(x_0, y(x_0)) h + \frac{h^2}{2} f'_x(x_0, y(x_0)) + f'_y(x_0, y(x_0)) f(x_0, y(x_0)) + O(h^3) \tag{1}
$$

В силу (1) справедливо

$$
y_1 \overset{[1]}{=}
y_0 + \frac{h}{2} \big[ f(x_0, y_0) + f(x_1, \bar{y}_1) \big] \overset{[2]}{=}
y_0 + \frac{h}{2} \big[ f(x_0, y_0) + f(x_0 + h, y_0 + h f(x_0, y_0)) \big] \overset{[3]}{=}
$$

$$
= y_0 + \frac{h}{2} f(x_0, y_0) + \frac{h}{2} \big[ f(x_0, y_0) + f'_x(x_0, y_0) h + f'_y(x_0, y_0) h f(x_0, y_0) + \tilde{O}(h^2) \big] \overset{[4]}{=}
$$

$$
= y_0 + h f(x_0, y_0) + \frac{h^2}{2} \big[ f'_x(x_0, y_0) + f'_y(x_0, y_0) f(x_0, y_0) \big] + \tilde{O}(h^3)
\tag{2}
$$

\[1] - расписали по [[Метод Эйлера с пересчетом#Процедура вычисления|процедуре]], пункт (2)

\[2] - подставили вместо $\bar{y}_1$ выражение из процедуры, пункт (1)

\[3] - разложили функции $f$ в ряд Тейлора по переменным $x$ и $y$ около точки $(x_0,y_0)$ и подставили в разложение точку, рассмативаемую ранее $(x_0 + h, y_0 + hf(x_0,y_0))$

\[4] - привели подобные, свойство $O$-больших

Рассмотрим разность выражений $(1)$ и $(2)$ Получим, с учетом того, что по [[Постановка задачи Коши для дифференциального уравнения 1 порядка|условию]] $y_0 = y(x_0)$:

$$
y(x_1) - y_1 = O(h^3) - \tilde{O}(h^3) = \hat{O}(h^3)
$$

Из равенства следует, что погрешность метода на одном шаге является величиной $O(h^3)$ -> Метод имеет 3 порядок точности на одном шаге.

В силу [[Замечание. Связь между порядком точности итерационного метода на одном шаге и на всей области решения|замечания]] метод имеет 2 порядок точности на всей области решения $[x_0, x_0 + X]$
