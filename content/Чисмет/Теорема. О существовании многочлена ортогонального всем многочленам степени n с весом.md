# Вводная

[[Теорема. Критерий максимальной алгебраической степени точности интерполяционной квадратурной функции с весовой функцией|критерий]] утверждает, что если ограничение на скалярные произведения у $n$ узлов выполняется, то [[Алгебраическая степень точности квадратурной формулы|алгебраическая степень точности]] у построенной [[Интерполяционная квадратурная формула с весовой функцией|формулы]] будет не меньше чем $2n+1$. Однако в критерии не доказывается, что узлы с наложенными ограничениями вообще найдутся. Покажем, что такие узлы существуют и найдем их

---

Чтобы узлы удовлетворялчи ограничению критерия (ортогональность $w_n$ любому многочлену степени $n$ и ниже c весом $p$)

$$
\int_a^b p(x)Q_n(x)\omega_n(x)dx = 0 \forall Q_n(x)
$$

$\omega_n(x) = (x - x_0)(x - x_1) \cdots (x - x_n)$

**Необходимо**, чтобы узлы были корнями $w_n$. То есть необходимо доказать существование соответсвующего многочлена

В общем виде $w_n$ может быть переписан как:

$$
G_{n+1}(x) = x^{n+1} + a_1 x^n + a_2 x^{n-1} + \dots + a_{n-1} x^2 + a_n x + a_{n+1} = \sum_{i=1}^{n+1}a_i\,x^{\,n+1-i}
$$

$a_i \in \mathbb{R}, i = 1,2,\dots,n+1$

Для того, чтобы многочлен $G_{n+1}$ был ортогонален с весом $p$ любому многочлену $Q_m(x)$ ($m \leq n$), необходимо и достаточно, чтобы

$$
\int_a^b p(x)G_{n+1}(x)x^m dx = 0 \quad \forall m = 0, 1, 2, \dots, n
$$

Равенства выше образуют неоднородную систему, содержащую $(n + 1)$ линейное алгебраическое уравнение относительно неизвестных $a_i \in \mathbb{R}$ ($i = 1, 2, \dots, n + 1$):

$$
0 =\int_a^b p(x)\,G_{n+1}(x)\,x^m\,dx
$$

$$
= \int_a^b p(x)\Bigl(x^{n+1} + \sum_{i=1}^{n+1}a_i\,x^{\,n+1-i}\Bigr)x^m\,dx
$$

$$
= \int_a^b p(x)\,x^{n+m+1}\,dx
  + \sum_{i=1}^{n+1}a_i\int_a^b p(x)\,x^{\,n+1-i}\,x^m\,dx
$$

$$
= \int_a^b p(x)\,x^{n+m+1}\,dx
  + \sum_{i=1}^{n+1}a_i\int_a^b p(x)\,x^{\,n+m+1-i}\,dx.
$$

В итоге **получаем систему** относительно $a_i, i = 1, \dots ,n+1$:

$$
\int_a^b p(x)\left(a_1 x^n + a_2 x^{n-1} + \dots + a_{n-1} x^2 + a_n x + a_{n+1}\right)x^m dx = -\int_a^b p(x)x^{n+m+1} dx
$$

$m = 0,1,\dots,n$

Из условия [[Постановка задачи вычисления определенных интегралов с весовой функцией|исходной задачи]] известны значения интегралов с весами

>Неоднородная система имеет единственное решение $\Leftrightarrow$ соответсвующая системе однородная система имеет лишь тривиальное решение (что-то из алгема)

$$
\int_a^b p(x)\left(a_1 x^n + a_2 x^{n-1} + \dots + a_{n-1} x^2 + a_n x + a_{n+1}\right)x^m dx = 0
$$

Достаточными условиями того, что рассматриваемая однородная система имеет лишь тривиальное решение являются следующие ограничения на $p$:

1) $p(x) \geq 0 \quad \forall x \in [a, b]$
2) $\displaystyle\int_a^b p(x)dx > 0$

Справедливость этого может быть доказана методом от противного.

Если предположить, что рассматриваемая однородная система имеет нетривиальное решение $(\bar{a}_1, \bar{a}_2, \dots, \bar{a}_{n+1}) \neq \mathbf{0}$, то суммирование

равенств (идею взяли из воздуха)

$$
\bar{a}_{n-m+1} \int_a^b p(x)\left(\bar{a}_1 x^n + \bar{a}_2 x^{n-1} + \dots + \bar{a}_{n-1} x^2 + \bar{a}_n x + \bar{a}_{n+1}\right)x^m dx = 0
$$

$\forall m = 0, 1, 2, \dots, n$

приводит к выражению:

$$
\int_a^b p(x)(\bar{a}_1 x^n + \bar{a}_2 x^{n-1} + \dots + \bar{a}_{n-1} x^2 + \bar{a}_n x + \bar{a}_{n+1})^2 dx = 0
$$

Которое противоречит ограничениям на $p$

# Формулировка

Таким образом, при наложении дополнительных условий на весовую функцию $p$ можно отыскать необходимые узлы, чтобы воспользоваться [[Теорема. Критерий максимальной алгебраической степени точности интерполяционной квадратурной функции с весовой функцией|критерием]]:

## Пусть

При решении [[Постановка задачи вычисления определенных интегралов с весовой функцией|задачи интегрирования функции с весом]] на весовую функцию наложены дополнительные ограничения

1) $p(x) \geq 0 \quad \forall x \in [a, b]$
2) $\int_a^b p(x)\,dx > 0$
3) $\exists \int_a^b p(x)x^k\,dx \quad \forall k = 0, 1, 2, \dots, 2n + 1$

## Тогда

Существует единственный многочлен

$$
G_{n+1}(x) = x^{n+1} + a_1x^n + a_2x^{n-1} + \dots + a_{n-1}x^2 + a_nx + a_{n+1},
$$

который ортогонален с весом $p$ любому многочлену $Q_n(x)$
