# Пусть

1) $y_{m-i} = y(x_{m-i})$ ($i = 1, \dots, k$) - известны **точные значения** [[Постановка задачи Коши для дифференциального уравнения 1 порядка|решения]], чтобы выполнить $m$-ый шаг метода (на точных данных (предыдущие [[Постановка задачи Коши для дифференциального уравнения 1 порядка|узлы]])

# Тогда

*Погрешностью [[k-шаговый разностный метод|k-шагового разностного метода]] на одном шаге* ($m$-ом) называется число:

$$
r_m = y(x_m) - y_m \overset{def}{=} y(x_m) - \sum_{i=1}^k \alpha_i y_{m-i} - h \sum_{j=0}^k \beta_j f_{m-j}
$$

$f_{m-j} = f(x_{m-j}, y_{m-j})$

> Иными словами, локальная погрешность - это то, насколько сдвинулось значение $y_m$ за один шаг при идеальных предыдущих данных

> Важно, что во входные данные не передается рассчитываемый $y_m$ (может появиться в формуле, если [[Неявный k-шаговый разностный метод|метод неявный]])
