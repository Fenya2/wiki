[[Интерполяционный кубический сплайн]] по определению задается на $[a,b]$ формулами:

$$
S(x) \overset{def}{=} p_i(x) = A_ix^3 + B_ix^2 + C_ix + D_i,\quad x \in [x_{i-1},x_i], i =  1,2,\dots,n
$$

Всего получается необходимо $4n$ параметров, чтобы задать [[Интерполяционный кубический сплайн|ИКС]]

Параметры должны быть подобраны так, чтобы удовлетворять следующим условиям:

1) $p_i(x_{i-1}) = y_{i-1},\, p_i(x_{i}) = y_{i},\, i = 1,\dots,n$ ([[Постановка задачи численной интерполяции|интерполяционное условие]])
2) $p'_i(x_i) = p'_{i+1}(x_i),\, i=1,\dots,n-1$ (требование непрерывности производных на "стыках" по определению [[Интерполяционный кубический сплайн|интерполяционного кубического сплайна]])
3) $p''_i(x_i) = p''_{i+1}(x_i),\, i=1,\dots,n-1$ (требование непрерывности вторых производных на "стыках" по определению [[Интерполяционный кубический сплайн|интерполяционного кубического сплайна]])

В сумме имеем $n + n + (n-1) + (n-1) = 4n - 2$ условия для определения значений $4n$ параметров

Для однозначного определения сплайна $S$ добавим к условиям на концы отрезка $[a,b]$ либо (краевые условия 1 типа):

1) $S'(a) = y'_0$ - Производная искомого сплайна в левом конце отрезка должна что?
2) $S'(b) = y'_n$

Либо (краевые условия 2 типа):

1) $S''(a) = y''_0$ - Производная искомого сплайна в левом конце отрезка должна что?
2) $S''(b) = y''_n$
