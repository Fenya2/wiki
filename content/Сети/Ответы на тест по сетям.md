# 197

_В чем недостаток маршрутизации между VLAN методом Router-on-a-Stick?_

1. Не поддерживает пакеты с тегами VLAN
2. Требует использования большего количества физических интерфейсов, чем традиционная маршрутизация между VLAN
3. ==Не масштабируется более чем на 50 VLAN==
4. Требует использования нескольких интерфейсов маршрутизатора, сконфигурированных для работы в качестве каналов доступа
