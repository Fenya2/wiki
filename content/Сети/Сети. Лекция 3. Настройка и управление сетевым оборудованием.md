---

Первая команда - настройка пароля на вход через консольный порт

line console 0 (настройка (режим глобальной конфигурации) консольной линии под номером 0 (обычно только одна))

password задает пароль

login говорит о том, что пароль нужно задать конкретно на той линии, на которой мы находимся

**Если ввели login, а пароль не задали - потеряли доступ к оборудованию**

enable secret задает пароль для привилегированного режима

13:42

![[Pasted image 20250320064619.png]]

Все пароли хранятся в файле конфигурации **в незашифрованном виде**. Чтобы защитить пароли от "Подглядывания через плечо", можно их зашфровать командой `service password-encryption`

Отмена команды (no) не расшифрует пароли, новые пароли после отмены команды будут в открытом виде

Единственная задача команды - сохранить пароли в зашифрованном виде в файле конфигурации. **Подключения не шифруют пароли при передаче** в этом случае

## "Джентльменский" набор команд

![[Джентельменский набор команд]]

Упрощают конфигурацию (вводятся перед началом настройки оборудования)

```
line console 0
logging synchronous (отключаем вывод логов работы оборудования в эту линию*)
no ip domain-lookup (если команда введеена неправильно, оборудование воспринимает его как доменное имя начаниект резолвить его в ip)
```
