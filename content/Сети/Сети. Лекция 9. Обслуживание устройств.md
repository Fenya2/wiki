## Коммутатор

### Светодиодные индикаторы

![[Pasted image 20250504134859.png]]

- Mode - настройка порта
	- Дуплексный режим
	- Скорость
	- PoE
- SYST - общее состояние коммутатора
	- горит зеленым - работает
	- желтый - питание есть, но работает с перебоями
	- выключен - нет питания
- RPC - состояние резервного источника питания
	- Горит зеленым - все хорошо
	- Горит желтым - питание неисправно
	- Мигает желтым - произошел сбой резервного питания
	- Не горит - есть проблемы
- STAT - статус порта (выбран и горит зеленым)
- DUPLX порты, находящиеся в дуплексном режиме
	- Зеленый - включен
	- Отключен - отключен (работает в полудуплексе)
- SPEED - скорость порта
- PoE (Power over Ethernet) - питание через Ethernet
	- Зеленый - подключено устройство, которое питается по Ethernet
	- Другие индикаторы - другое поведение

## Резервное копирование конфигурации

![[Pasted image 20250504141518.png]]

Бекап конфигурации

Способ 1:

```c
show running config
```

Скопировать вывод и вставить

Есть нюансы с восстановлением (удаление лишних символов), но способ рабочий

Способ 2:

Скопировать конфигурацию на удаленный хост

```cisco
copy running-config tftp
```

2 аргумент - протокол

Далее в интерактивном режиме:

- Адрерс удаленного хоста

Копирование на флешку

```cisco
copy running-config usbflash0
```
