# Общие сведения о выборочном методе

![[Репрезентативная выборка]]

![[Случайная выборка]]

[[Случайная выборка]] чаще всего является [[Репрезентативная выборка|репрезентативной]]

Прочие способы выборки:

- Механический (*каждый $n$-ый из [[Генеральная совокупность|генеральной совокупности]]*)
- Серийный (выбираем *сплошными* сериями по $k$ штук )
- По типу
	- Равномерный
	- Пропорциональный
- По повторяемости
	- Без повторов
	- [[Повторная выборка|Повторный]]

# Ошибки, связанные с неудачно выбранной [[Выборка|выборкой]]

Разделяются на 2 категории:

- Репрезентативные
	- Систематические (нарушена случайность отбора при составлении [[Выборка|выборки]])
	- Случайные (из-за неудачной выборки, несмотря на ее [[Случайная выборка|случайный]] характер)
- Регистрационные (при фиксации данных допустили неточность)
