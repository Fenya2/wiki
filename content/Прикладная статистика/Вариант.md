# Вариант

Разобьем элементы [[Выборка|выборки]] на группы по признаку, что элементы группы имеют одно значение. Из каждой группы выберем элемент-представитель. Значение элемента-представителя называется *вариантом*

Обозначается $x_i$

# Частота варианта

Число элементов в каждой группе называется *частотой варианта*

Обозначается $n_i$
