*LR-анализатором* [[Расширенная грамматика|расширенной грамматики]] $G$ называется таблица *Action/GOTO*, в которой строки соответствуют состояниям какого-то [[Детерменированный конечный автомат|ДКА]] $A$, построенного по грамматике $G$, столбцы символам из $\Sigma \cup \Gamma \cup \{\dashv\}$

|           | ACTION   |          | GOTO     |
| --------- | -------- | -------- | -------- |
| Состояние | $\Sigma$ | $\dashv$ | $\Gamma$ |
|           |          |          |          |
|           |          |          |          |

Множество действий **ACTION** может быть 4 видов:

- $\sqrt{}$ - *допуск*
- $\leftarrow q$ - *перенос* ($q$ - название состояния в соответсвующем [[LR(0)-автомат|LR(0)-автомате]], по которому строится анализатор)
- $\otimes n$ - *свертка* (по правилу с номером $n$)
- $err<n>$ - ошибка анализа с номером $n$ (опционально)

В столбцах нетерминалов **GOTO** стоят состояния автомата $A$

> Вид автомата $A$ зависит от вида *анализатора*, который требуется построить

> Также так могут называть [[Детерменированный автомат с магазинной памятью|автомат с магазинной памятью]], который работает по правилам, опирающимся на таблицу анализатора
