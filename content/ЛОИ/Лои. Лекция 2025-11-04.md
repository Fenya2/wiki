# Проведение [[Семантический анализ|семантического анализа]] для [[S-атрибутная грамматика|S-атрибутных грамматик]]

**Замечание:** естественный способ вычисления [[Синтезируемый атрибут|синтезируемых атрибутов]] "снизу-вверх" совпадает с восстановлением "снизу-вверх" [[Дерево вывода слова|дерева вывода]] при [[Лои. Лекция 2025-10-07|LR-анализе]]. Следовательно, [[Синтаксический анализ|синтаксическй анализ]] можно совместить с [[Семантический анализ|семантическим анализом]]

То есть, если имеем [[Бесконфликтный LR-анализатор|бесконфликтный]] [[LR-анализатор]] для [[S-атрибутная грамматика|S-атрибутной грамматики]], то можем за одну работу анализатора (проведение [[Синтаксический анализ|синтаксического анализа]]) заодно вычислить значения всех [[Атрибут грамматического символа|атрибутов]] с использованием того же [[LR-анализатор|анализатора]] (то есть провести [[Семантический анализ|семантический анализ]])

Чтобы провести вычисление атрибутов при работе [[LR-анализатор|анализатора]], рядом с основным стеком заводится второй стек $STACK$, в котором будут храниться дополнительная информация для вычисления значения всех [[Атрибут грамматического символа|атрибутов]] всех [[Грамматический символ|грамматических символов]]. В нем будут храниться значения атрибутов, необходимые для вычисления.

## Модификация алгоритма работы [[LR-анализатор|LR-анализатора]]

Считаем, что для каждого [[Грамматический символ|грамматического символа]] в рассматриваемой [[S-атрибутная грамматика|S-атрибутной грамматике]] задан только один атрибут

- будет выполняться операция [[LR-анализатор|свертки]] по правилу $P = A \to \alpha$

```java
STACK.push(f(parameters))
```

# Пример проведения [[Семантический анализ|семантического анализа]] для [[S-атрибутная грамматика|S-атрибутных грамматик]]

Распишем подробно, как происходит [[Семантический анализ|семантический анализ]] числового выражения $w = (2+4)+5*3$, которому соответствует [[Слово|слово]] $w=(x+x)+x*x$ в [[Расширенная грамматика|расширенной грамматике]]:

$$
G = \{ E' \to E, E \to E + T | T, T \to T * F | F, F \to (E) | x \}
$$

Для этого пронумеруем ее [[Порождающая грамматика|правила вывода]]:

| Номер | [[Порождающая грамматика\|Правило вывода]] |
| ----- | ------------------------------------------ |
| 0     | $E' \to E$                                 |
| 1     | $E \to E + T$                              |
| 2     | $E \to T$                                  |
| 3     | $T \to T * F$                              |
| 4     | $T \to F$                                  |
| 5     | $F \to (E)$                                |
| 6     | $F \to x$                                  |

Построим [[LR-анализатор]] для этой грамматики (не важно какой именно. Главное, что он [[Бесконфликтный LR-анализатор|бесконфликтный]])

| $E_1$    | $\leftarrow +$ |                |                |                |                | $\sqrt{}$    |       |       |       |
| $T_1$    | $\otimes 2$    | $\leftarrow *$ |                |                | $\otimes 2$    | $\otimes 2$  |       |       |       |
| $F_1$    | $\otimes 4$    | $\otimes 4$    |                |                | $\otimes 4$    | $\otimes 4$  |       |       |       |
| $($      |                |                | $\leftarrow x$ | $\leftarrow ($ |                |              | $E_2$ | $T_1$ | $F_1$ |

Зададим [[Атрибут грамматического символа|атрибуты]] ([[Синтезируемый атрибут|синтезируемые]]) и [[Семантическое правило|правила]] для их вычисления:

| Номер правила | [[Порождающая грамматика\|Правило вывода]] | [[Семантическое правило]] |
| ------------- | ------------------------------------------ | ------------------------- |
| 1             | $E \to E_1 + T$                            | $E.ptr = E_1.ptr + T.ptr$ |
| 2             | $E \to T$                                  | $E.ptr = T.ptr$           |
| 3             | $T \to T_1 * F$                            | $T.ptr = T_1.ptr * F.ptr$ |
| 4             | $T \to F$                                  | $T.ptr = F.ptr$           |
| 5             | $F \to (E)$                                | $F.ptr = E.ptr$           |
| 6             | $F \to x$                                  | $F.ptr = x.ptr$           |

> $x.ptr \overset{def}{=} x.lexval$

В случае [[Слово|слова]] $w$ атрибут $lexval$ имеет [[Грамматический символ|символ]] $x$ и может принимать значения $2,3,4,5$
