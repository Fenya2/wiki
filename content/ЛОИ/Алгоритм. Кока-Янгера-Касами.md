# Вход

- [[Контекстно-свободная грамматика|КС-грамматика]] $G=(\Gamma, \Sigma, P, S)$ в [[Нормальная форма Хомского|нормальной форме Хомского]]
- [[Слово]] $w \in \Sigma^+$

# Выход

- [[Выводимость слова|Вывод слова]] $w$ в [[Порождающая грамматика|грамматике]] $G$, если $w \in L(G)$ и сообщение об ошибке в противном случае

# Алгоритм

## Заполнение таблицы

Обозначим $n=|w|$

По сути является алгоритмом динамического программирования. В процессе работы будем заполнять полу-таблицу размерности $n \times n$. Причем строки такой таблицы, полагаем, нумеруются с $0$, а столбцы с $1$

| Номер строки i | 1   | 2   | $\dots$ | n   |
| -------------- | --- | --- | ------- | --- |
| $0$            |     |     |         |     |
| $1$            |     |     |         |     |
| $\dots$        |     |     |         |     |
| $n-1$          |     |     |         |     |

В ячейках таблицы будем записывать множества [[Порождающая грамматика|нетерминалов]] $\Gamma_{i,j}, j=1,\dots, n, i=0,\dots,n - j$

### Заполнение 1 столбца таблицы $j=1$

[[Порождающая грамматика|Нетерминалы]] $Г_{i,1}, i=0, \dots, n-1$ вычисляются следующим образом:

$$
X \in \Gamma_{i,1} \Leftrightarrow \exists (X \to w_i \in P)
$$

где $w_i$ - $i$-ый символ цепочки w

### Заполнение 2 столбца таблицы $j=2$

[[Порождающая грамматика|Нетерминалы]] $Г_{i,2}, i=0, \dots, n-2$ вычисляются рекурентно:

$$
X \in \Gamma_{i,2} \Leftrightarrow \exists (X \to X_1X_2 \in P), X_1 \in \Gamma_{i,1}, X_2 \in \Gamma_{i+1, 1}
$$

### Заполнение оставшихся столбцов таблицы $j>2$

[[Порождающая грамматика|Нетерминалы]] $Г_{i,j}, j>2$ вычисляются рекурентно:

$$
X \in \Gamma_{i,j} \Leftrightarrow \exists k,m > 0, k+m = j : \exists (X \to X_1X_2 \in P), X_1 \in \Gamma_{0,k}, X_2 \in \Gamma_{k, m}
$$

## Выводы

[[Слово]] $w$ принадлежит [[Выводимость слова|Языку]] $L(G)$, если $\Gamma_{0, n} = \{ S \}$. В противном случае не принадлежит. Алгоритм выбрасывает ошибку
