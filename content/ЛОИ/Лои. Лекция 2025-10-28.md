# [[Синтаксическое дерево|Синтаксические деревья]]

[[Дерево вывода слова|Деревья вывода]] могут быть довольно громоздкими для последующей с ними работы. Существуют методы их эквивалентного "сжатия".

Один из способов "Сжатия" [[Дерево вывода слова|дерева вывода]] - построение [[Синтаксическое дерево|синтаксического дерева вывода]] по исходному [[Дерево вывода слова|дереву вывода]]. Метод применим, если исходная грамматика, по которой построено [[Дерево вывода слова|дерево вывода]] является [[Операторная грамматика|операторной грамматикой]]
